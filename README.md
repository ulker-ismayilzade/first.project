# Python To-Do List Application
A simple and efficient command-line interface (CLI) application built with Python to manage daily tasks.

## Features
- Add, view, and delete tasks.
- Data persistence using file handling (TXT/JSON).
- User-friendly error handling for input validation.

## Technical Skills Demonstrated
- Python Fundamentals (Lists, Loops, Functions).
- File I/O operations.
- CLI Application Development.
